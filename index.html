Prerequisites:

A Computer, Free Time, Dedication, Willingness to Learn

Objectives:

In the course or this class you will accomplish the following:


  • Gather a basic understanding of HTML5 and its uses
  • Gather a basic understanding of CSS3 and its uses
  • You will create basic webpages and web applications showing proper usage of each
  • Gather a basic understanding of Javascript and its uses
  • Learn to use advanced Front End Frameworks like ReactJS or Bootstrap
